## Dev Notes
- **Modelo Usado**: `gemini-embedding-001` (modelo mais recente, disponível na cota Free do AI Studio — 100 RPM / 1000 RPD).
- **Sem Banco Externo**: Gravamos os vetores direto no Array do `vectors.json`. Um modelo de NLP de 768 dimensões para ~100 artefatos vai gerar um arquivo menor que 2MB (Gzip de ~300KB), o que é totalmente executável e cacheável em Memória pelo Node.js/Vercel no Runtime final do site. 
- **Invalidação Transparente**: Um Hash curto B64 do texto bruto é salvo junto do Array Numérico para que a pipeline execute de modo diferencial em 4 segundos nas madrugadas.
