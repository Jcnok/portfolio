## Story
Como administrador do portfólio,
Eu quero que uma GitHub Action varra minha pasta do Google Drive em busca de novos certificados em PDF, transcrevendo-os via Gemini 1.5 Flash Vision,
Para que meus conhecimentos comprovados virem dados legíveis para a I.A. sem eu precisar digitar nada.

## Dev Notes
- **Zero dependências externas** para autenticação Google: implementação JWT pura usando `crypto.createSign` nativo do Node.js.
- **Ingestão incremental**: compara `driveFileId` dos PDFs do Drive com os já registrados em `certificates.json`. Só processa novos.
- **Rate limiting**: 5 segundos de sleep entre cada chamada ao Gemini Vision (Free Tier: 15 RPM).
- **Fallback**: Se a transcrição Gemini falhar para um PDF, o arquivo é registrado com dados mínimos para não ser reprocessado.
- **Modelo**: `gemini-1.5-flash` (Vision) — suporta `inlineData` com `mimeType: application/pdf` nativamente.
